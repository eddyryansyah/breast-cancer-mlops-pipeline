# Breast Cancer Awareness and Tumor Classification

This Hugging Face Space provides a public demo for the Breast Cancer MLOps Pipeline project.

The demo includes:

1. Breast health awareness guidance for unusual breast changes.
2. Machine learning sample prediction using numerical tumor feature samples from the Breast Cancer Wisconsin Diagnostic dataset.

## Responsible Use

This application does not provide medical diagnosis. The awareness section is intended to encourage users to seek medical consultation when they notice unusual or persistent breast changes. The machine learning model classifies numerical tumor feature samples and does not analyze personal symptoms directly.
